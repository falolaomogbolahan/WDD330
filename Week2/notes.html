  1. Hoisting means moving variables to the top of the scope.

Things to remember

> Use let instead of var because let allows block scope which var doesnt. var may be seen in older code because it's an older version when writing variable name use either camelCased or underscore_separated

In the following code, changing d.value also changes c.value because d and c reference the same object
const c = { value: 1 };
let d = c; // c.value = 1, d.value = 1
d.value = 2; // c.value = 2, d.value = 2
  • String functions
    • .toUpperCase() : string
    • .toLowerCase() : string
    • .charAt(index) : char
    • .indexOf(character) : number
    • .lastIndexOf(character) : number
    • .startsWith(character) : bool
    • .endsWith(character) : bool
    • .concat('string', 'anotherString') : string
