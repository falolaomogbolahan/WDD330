Ajax

Ajax is a technique that allows wb pages tocommunicate asynchronously with a server, and it dynamically updates web pags without reloading. this enables data to be sent and received in the background as well as portions of a page to be updated in response to user events, while the rest of the program continues to run.

The data can be sent in many forms, but its usually in JSON.

Ajax can be used for making partial page udates without having to do a full page reload.

Ajax can be used fo communicating with extenal APIs, it can also be used to fetch API.
