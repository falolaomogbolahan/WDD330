This week I studied objects, properties and methods, Objects are said to be a seld-conatined set of functions and related values. it is used to keep any related information and functionality together in one place.

Objects are variables, but they can contain many values. each variables are seperated by a comma, and all its properties are wrapped in curly braces, Behold a simple example

const metahuman = {type:"villain", Power:"Invisibilty", name:"trans"};

Each property is a key-value pair, seperated by commas and the values can be accessed by simply typing metahuman.name. this would target the name of the metahuman and the variable would be trans. it can also be accessed using the square bracket notation, metahuman ['name']. the same thing would happen.

IF YOU TRY TO ACCESS A PROPERTY THAT DOESNT EXIST, undefined will be returned.

The in operator can be used to check wheather an object has a particular property,
